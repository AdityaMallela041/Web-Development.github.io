Web Development

HTML

HyperText Markup Language - The standard markup language for creating web pages.

CSS

Cascading Style Sheets - Used for styling HTML documents and enhancing the look and feel of web pages.

JavaScript

A high-level, interpreted programming language that enables interactive web pages and is an essential part of web development.

React

A JavaScript library for building user interfaces, particularly for single-page applications where UI updates are frequent.

Node.js

An open-source, cross-platform JavaScript runtime environment that executes JavaScript code server-side.

Python

A versatile and powerful programming language often used for web development, data science, and automation.

Angular

A TypeScript-based open-source web application framework led by the Angular Team at Google.

Vue.js

A progressive JavaScript framework used for building user interfaces, focusing on the view layer.

PHP

A server-side scripting language designed for web development but also used as a general-purpose programming language.
